Bem vindo!

Para realizar a integração de arquivos Excel com o sistema, siga os passos abaixo:

  1. Configurar API: Na opção "Conexões API", configure a API que deseja utilizar.
  2. Relacionar campos: Na opção "Modelos", faça o mapeamento entre os campos da sua planilha e os parâmetros configurados.
  3. Upload de Arquivos: Após a configuração, vá para a opção "Upload de Arquivos", envie sua planilha e selecione o modelo correspondente.
  4. Verificações: Use as opções a seguir para acompanhar o processamento dos dados:
    • "Consulta de Erros": Exibe inconsistências encontradas na planilha que impediram o envio para a API.
    • "Consulta Controle API": Exibe o status dos dados que foram convertidos e enviados com sucesso.

Estamos aqui para facilitar o seu trabalho e garantir que suas integrações sejam eficientes e sem complicações!
